# Transcripts and Conversation Source List

## 访谈与对话素材

- 录音/视频/播客字幕：阿尔法公社、李笑来自媒体访谈、加密圈交流活动。
- 演讲实录：公开课程、线下分享、投资主题演讲。
- 长对话文本：与其他投资人、创业者的深入问答。
- 若有《我也有话要说》《人人都能用英语》等文字作品的音频或访谈版本，也可作为对话素材参考。

## 操作建议

- 将已有 SRT/TTML/文本文件放入本目录，并命名为 `YYYYMMDD-主题-来源.ext`。
- 若只有视频链接，先用 `scripts/download_subtitles.sh` + `scripts/srt_to_transcript.py` 处理。
- 如果从李笑来个人主页获取到演讲或访谈页面，将链接写入本文件以便后续转录。
- 本文件用于记录已搜集的对话素材来源、URL 和摘要关键词。
