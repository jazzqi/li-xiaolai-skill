# extraction-framework

## 提炼方法论

本文件记录用于从调研材料中提取心智模型、决策启发式和表达DNA的原则。

### 1. 心智模型三重验证

- 跨域复现：同一模型是否出现在不同类型素材中（著作、对话、决策）。
- 生成力：该模型是否能推断出该人物对新问题的立场。
- 排他性：该模型是否具有一定的个体识别度，而不是通用的智识表达。

### 2. 决策启发式标准

- 形态：应为“如果…则…”或“先…再…”的快速规则。
- 证据：至少有一个公开案例或说法支持。
- 适用场景：说明它适合解决哪类判断问题。

### 3. 表达DNA提取

- 句式偏好：短句/长句、转折、反向表达。
- 词汇特征：高频词、禁忌词、专属术语。
- 风格特征：结论先行、坦诚边界、反向拆解。

### 4. 质量自检清单

- 是否有3-7个心智模型，每个都注明来源。
- 是否有5-10条决策启发式。
- 是否给出了表达DNA的可操作规则。
- 是否包含明确的诚实边界与内部张力。
- 是否标注信息截止时间和来源类型。
