# 06-timeline · 李笑来人生阶段与时间线（重建版）

## 方法说明与证据边界

- **目标**：为最终 Skill 的「身份卡 / 时间线 / 阶段变化」提供一份可复用、可溯源、尽量少神话化的人生年表。
- **取材原则**：优先使用李笑来本人官网与其公开文本（GitHub/官网书稿/课程入口页），必要时再用可核对的外部出处；本稿**不使用**知乎、微信公众号文章、百度百科/知道。
- **证据现实**：李笑来的许多关键节点，最可靠来源恰恰是他自己的书与官网说明。因此下文会明确区分：
  - **高可信**：官网/GitHub 作品仓库、本人书稿中的一手自述；
  - **中可信**：仍为本人自述，但带有明显回忆性、辩护性或后见之明色彩；
  - **待补证**：当前第一手材料能证明大概方向，但不能精确到某年某月。
- **一个重要提醒**：本稿有意避免写入一些在中文互联网被大量复述、但我尚未在第一手材料里看到硬证据的细节（例如常见的出生年份说法）。最终 Skill 若要放进「出生于某年」之类信息，建议另补出版社作者简介、法人/简历页或本人明确自述作为佐证。

---

## 一页结论

李笑来的公开人生，不宜只概括成“币圈投资人”。按照第一手材料，更稳妥的分法是：

1. **大学毕业后的谋生/销售阶段**：先做销售与批发，经历家庭变故与积蓄归零；
2. **新东方英语教师与考试写作阶段**：约 2000 年末进入新东方，建立“会讲、会拆解、会写教材”的公共人格；
3. **从英语老师到方法论作者阶段**：2007 年起写《把时间当作朋友》，从“教语言”上升到“教方法、教心智”；
4. **比特币学习者—持有者—布道者阶段**：2011 年进入比特币，经历暴涨暴跌，形成其后全部 crypto 叙事的实战底色；
5. **区块链投资人与高曝光争议阶段**：2015Q4 起批量投 MBA 项目，2017–2018 年达到曝光高点，也在 2018 年遭遇“录音/韭菜”式形象坍塌；
6. **收缩投资边界、转向课程/社群/长期主义产品阶段**：2019 年后官网明确写“更多在讲课”，2018/2019/2022 连续收缩投资边界，后期定位越来越像“方法论讲授者 + 社群经营者 + 定投/思考/财富课程产品经理”。

如果只从 2011 年比特币开始写，李笑来会被误写成“赶上时代红利的币圈人物”；但按其第一手材料看，更深的母线其实是：**教师 → 方法论作者 → 风险承担者 → 争议人物 → 收缩边界后的课程/社群主理人**。

## 年代序列（按证据强弱排序）

### 1990 年代中期（可确认曾接触北京/新东方，但不是职业节点）

- 在《人人都能用英语》中，李笑来回忆自己“那一年夏天”去北京找同学装 486 电脑，路过新东方旧址；随后多年后才真正进入新东方。[S2]
- **意义**：这不是职业起点，但说明“新东方/出国考试培训”在他正式加入前，就已进入其认知背景。
- **可信度**：中（个人回忆，具体年份靠文内“1994 年”回忆链推断）。

### 大学毕业后至 2000 年前后：会计专业毕业 → 做销售/批发 → 家庭变故后需要稳定工作

- 《人人都能用英语》与《自学是门手艺》互相印证：他大学是**会计专业**；毕业后并未真正做会计，而是做过几年销售/批发；后来在父亲养病期间花光积蓄，急需稳定收入。[S2][S4]
- 《自学是门手艺》甚至把自己概括为：“一个末流大学的会计专业毕业的人不得已去做了销售”。[S4]
- **意义**：这段经历解释了他后来为什么极重视“生产能力、销售能力、现金流、安全边界”。这不是后来投资阶段才冒出来的主题，而是早年谋生经验留下的底色。
- **可信度**：高（多份第一手文本交叉）。

### 2000 年年底（或 2001 年初前后）：赴北京，进入新东方；随后教书约 7 年

- 《人人都能用英语》写得很具体：**“2000 年年底”**，他到北京、报新东方的班、考托福/GRE，再去应聘，**“而后一口气那里教了七年”**。[S2]
- 这段时间内，他在托福/GRE/GMAT 等出国考试培训中建立起名师身份，也开始写与考试有关的教材。[S2][S3]
- **意义**：这是李笑来“公开人格”的真正起点。其后无论写时间管理、讲自学、谈投资，底层动作都仍然带有课堂感：拆概念、重定义、反复强调常识、用直白句式降维输出。
- **可信度**：高。
- **不确定处**：如果按“2000 年底进入、教了七年”机械推算，离开新东方大约在 2007–2008 年之间；但当前第一手材料没有给出更精确离职日期。

### 2003：出版《TOEFL 核心词汇 21 天突破》

- 《人人都能用英语》与《自学是门手艺》都明确提到：**2003 年**《TOEFL 核心词汇 21 天突破》出版。[S2][S4]
- 官网 README 也把这本书列为其早期出版作品之一。[S1]
- **意义**：他从“教师”转向“可规模化复制的内容生产者”。这是后来所有书、课程、专栏、社群产品化的早期样板。
- **可信度**：高。

### 2007：开始写《把时间当作朋友》的原始网志；主题从“语言教学”升到“方法/心智”

- 《把时间当作朋友》第三版前言明确写：这些文字**“2007 年年初开始陆续地写”**，最初是写给学生的，文章系列标题从“管理我的时间”变成“把时间当作朋友”。[S3]
- 同一前言还说：这些文字最初并无出版计划，只是先发在网志上。[S3]
- **意义**：这是李笑来一生中极重要的抽象跃迁——从“教英语”升级为“教如何学习、如何思考、如何面对时间与现实”。
- **可信度**：高。

### 2009-06：**《把时间当作朋友》正式出版**

- 第三版前言明写：**“2009 年 6 月《把时间当作朋友》正式出版发行。”**[S3]
- 同文还回顾：首印 8000 册，一周售罄；此后一年半重印 12 次。[S3]
- **意义**：李笑来第一次从“新东方名师”进化为“方法论作者”。在公众形象上，这一阶段的他更像“认知升级/个人成长作者”，而不是投资人。
- **可信度**：高。

### 2010：第二版修订；2010 年底形成第三版重写文本

- 《把时间当作朋友》第三版前言写：**2010 年年初**修订文字、更新例子，算第二版；**2010 年年底**觉得必须重写，于是形成第三版文本。[S3]
- **意义**：说明《把时间当作朋友》不是“一次性爆款”，而是他反复迭代的母文本。
- **可信度**：高。
- **不确定处**：第一手材料在这里更清楚地给出了“写作/修订”时间，而不是每一版印刷上市时间；若最终 Skill 要写“第三版于某年上市”，最好补出版社记录。

### 2011-03 至 2011-06：第一次见到 Bitcoin，并在 4–6 月间开始买入

- 《韭菜的自我修养》中有极关键的自述：**“2011 年 3 月初，我在 Twitter 上第一次见到 Bitcoin 这个词”**；等到四月底账号弄好开始买，价格已四美元多；买完第一批 2100 个时，均价约 6 美元；六月初最高涨到 32 美元。[S5]
- 同段又写到，随后经历长达二十多个月的深度回撤与解套过程。[S5]
- **意义**：这不是普通“早期知道比特币”的叙述，而是他后来自我定位的核心分水岭：从“解释世界的人”变成“真金白银下注并承受波动的人”。
- **可信度**：高（第一手自述）；但属于回忆性叙述，精确成交细节无法独立核验。

### 2013：在线版《人人都能用英语》代表其“英语教学总结期”达到成熟

- 《自学是门手艺》回顾性写道：后来陆续写了很多内容，包括**“只有在线版发行的《人人都能用英语》（2013）”**。[S4]
- **意义**：这可被视为他“英语老师身份”的系统性收束。到这里，他已经不只是教应试，而是把语言学习、成长方法、长期投入合成一套可传播的框架。
- **可信度**：高。

### 2014 年之前：已投资比特股、云币网，并长期维护“比特沙”项目

- 《INBlockchain 的开源区块链投资原则》写得很明确：**“2014 年之前，我投资了比特股、云币网，并且自己长期维护一个项目，比特沙，一个比特币银行。”**[S6]
- **意义**：到这个节点，他已不只是币圈散户或长期持币者，而是开始进入“项目、平台、基础设施”层面的参与。
- **可信度**：高。

### 2015 年 Q4：开始批量投资所谓 MBA（Meaningful Blockchain Applications）项目

- 同一份《INBlockchain 的开源区块链投资原则》写道：**“从 2015 年最后一个季度开始，我和我的团队开始批量投资 MBA 项目。”**并列出 Sia、Steem、Zcash、QTUM、EOS 等。[S6]
- **意义**：这是李笑来从“个人比特币受益者”转向“机构化配置者/生态投资人”的明确标志，也是其公共形象从“作者/老师”被覆盖为“币圈投资人”的关键阶段。
- **可信度**：高。

### 2017：财富话语与区块链投资原则公开化

- 《自学是门手艺》回顾：**《通往财富自由之路》（2017）**与在得到 App 上开专栏有关，是其此前“学习学习再学习”的重构扩充。[S4]
- 《INBlockchain 的开源区块链投资原则》正文以 **“2017 年 6 月”** 为时点讨论市场与 ICO 环境，并明写“目前，我们正在积极参与全球的 ICO 项目”。[S6]
- **意义**：2017 年的李笑来，已经同时站在两条叙事线上：
  1. 面向大众的“财富自由/长期积累”作者；
  2. 面向币圈的“区块链项目投资人/原则输出者”。
- **可信度**：高。

### 2018：录音争议爆发，个人公共形象发生剧烈扭转；《韭菜的自我修养》成为系统回应文本

- 《韭菜的自我修养》前言直接把写作动机绑在争议上：他说此前有一次**私下谈话被偷录并传播**，导致网上大量文章把他与“割韭菜”绑定，甚至使其成为“全民公敌”；他强调自己原本并不使用“韭菜”这个词。[S5]
- 同书明确自报年份为 **2018**。[S5]
- **意义**：这是李笑来 public image 的最大断裂点。此前公众可以把他理解为“名师、作者、投资人”；此后更大范围的中文互联网公众，会优先以“币圈、割韭菜、争议人物”来记忆他。
- **可信度**：高（至少可确认：他本人把 2018 的书直接当作对录音舆论的回应文本）；
- **但要保留边界**：关于录音外部传播链、媒体报道时间点、当时市场环境与具体发酵路径，仅凭第一手材料无法完整重建，本稿不做超证据延展。

### 2018-10-31：发布比特币白皮书中英对照译本

- 官方仓库中的《比特币：一种点对点的电子现金系统》页面明确署名：**“中文翻译：李笑来……2018.10.31”**。[S7]
- **意义**：在 2018 舆论风暴之后，他并没有退出加密叙事，而是继续以“解释者/翻译者/教育者”身份参与比特币话语生产。
- **可信度**：高。

### 2018 年之后：官网明确写明“不再参与早期项目投资”

- 官网 README 的“投资范围”部分直写：**“2018 年之后，不再参与早期项目投资”**。[S1]
- **意义**：这是一个非常重要的、容易被忽视的边界收缩信号。很多外界叙事仍停留在“李笑来 = 币圈早期项目投资人”，但其官网自述已经把这个身份主动收窄。
- **可信度**：高。

### 2019-02-27：完成《自学是门手艺》初稿；2019 年后更偏向“讲课”而非“写作”

- 《自学是门手艺》末尾署名：**“初稿完成于 2019 年 2 月 27 日”**。[S4]
- 官网 README 又写：**“2019 年后，我更多在‘讲课’，而不是在‘写作’……”**[S1]
- 同一 README 还写：**“2019 年之后，不再管理新的基金。”**[S1]
- **意义**：这是第二个大转弯：从高波动市场参与者，转向“课程、社群、方法论”的稳定经营；也是他把“长期主义、定投、自学、生产、销售、投资”串成一整套产品体系的起点。
- **可信度**：高。

### 2019 年后：富足人生践行群 / 人生最重要的一课 / 多个“真相”课程，构成其后期产品矩阵

- 官网 README 列出：**富足人生践行群**（与“定投人生课堂”合并）、**人生最重要的一课**、以及“好的家庭教育”下的多个主题课：**学习的真相、教练的真相、思考的真相、财富的真相、多语的真相、专注的真相**等。[S1]
- 同一处写“富足人生践行群”的核心关键字为：**自学、生产、销售、投资、追求**。[S1]
- **意义**：这说明他后期不只是“继续写书”，而是在做一套高度产品化、社群化、陪伴式的认知课程系统。
- **可信度**：高。
- **不确定处**：官网能确认这些课程/配套书籍的存在，但**并未在同一页面给出每本书/每门课的精确首发年份**。例如《财富的真相》《思考的真相》的准确出版时间，仍建议后补出版社页面或版权页。

### 2022 年之后：继续收缩投资边界，官方表述变为“不再做二级市场外的投资”

- 官网 README 明写：**“2022 年之后，不再做二级市场外的投资……”**[S1]
- **意义**：如果说 2018 和 2019 的变化还可以解释为“周期切换后的暂时保守”，那么 2022 这条边界则更像长期再定位：李笑来有意识地退出高接触度、重关系、重项目筛选的投资动作。
- **可信度**：高。

### 近年（至少截至 2025）：仍持续经营内容分发渠道，官方站点保持更新，不是“消失”状态

- 官方仓库保留 `wxvideo.md`，明确引导读者在微信视频号搜索“笑来”，并展示大量嵌入视频。[S8]
- 官方作品 README 仍持续维护；GitHub 仓库近端提交记录显示，**2025-08** 及之后该站点仍有更新提交。[S9]
- **意义**：这能支持一个重要判断：李笑来近年的姿态不是“彻底退场”，而是**从高曝光的币圈中心舞台，转向自有渠道、课程体系和更可控的内容分发**。
- **可信度**：高（对“站点与内容仍在更新”这一点）。
- **不确定处**：第一手公开材料能证明“仍在更新/仍在发内容”，但未必足以重建其近两年全部活动强度、商业规模或影响范围。

## 关键转折点（worldview / public image）

### 转折 1：从“会计/销售出身的谋生者”到“英语教师”

这一步的本质不是名气，而是能力结构的变化：
- 从直接卖货、赚差价，转向“讲解、拆解、说服、训练”；
- 这使他后来所有作品都带有强烈的“讲师底色”。

### 转折 2：2007 年从“教语言”升到“教方法”

《把时间当作朋友》不是副业写作，而是李笑来人格的一次抽象上移：
- 语言只是载体；
- 真正开始输出的是心智、时间、学习、现实感。

从这里开始，他不再只是“新东方老师李笑来”，而是“会给成年人重装操作系统的人”。

### 转折 3：2011 年进入比特币，开始承担真实下注后果

在此之前，他的角色主要是解释者；
在此之后，他成了**参与者**。

这一步极关键，因为李笑来后来的很多锐利判断、轻蔑感、反大众情绪、对市场周期与认知错误的执念，都离不开这轮早期暴涨暴跌的经验。

### 转折 4：2015Q4–2017，从个人持有者变成机构化投资者

这是公众形象被大规模改写的阶段：
- 他不再只是“早期买了比特币的人”；
- 而是开始和基金、项目、ICO、交易平台、投资原则绑定。

这也意味着：他的名字从此与更大的收益想象、也与更大的道德风险想象捆在一起。

### 转折 5：2018 录音/“韭菜”争议，把他的公众身份从“老师/作者/投资人”压扁成“争议人物”

《韭菜的自我修养》本身，就是危机后的自我叙事重建工程。

需要特别记住的是：
- **他的自我理解**：概念被曲解，他是在澄清；
- **公众残留印象**：很多人从此只记得“割韭菜”标签。

最终 Skill 必须同时容纳这两个层面，否则会失真。

### 转折 6：2019 / 2022 之后，主动收缩投资边界，回到课程/社群/方法论产品

官网连续三条边界说明（2018、2019、2022）很重要：
- 不再投早期项目；
- 不再管新基金；
- 不再做二级市场外投资。

这说明后期李笑来不是“继续外扩”，而是**主动收口**。这与很多创业/投资叙事里“永远扩张”的姿态不同，也说明他的晚近定位更偏向：
- 自有渠道内容输出；
- 小范围高粘性社群；
- 以“自学—生产—销售—投资”为轴的长期生活方式课程。

## 对最终 Skill 最有价值的 6 个阶段概括

### 阶段 A：谋生与销售底色期（大学毕业后—2000）

关键词：会计专业、销售、批发、家庭压力、现金流意识。  
这一阶段解释他为什么长期警惕“没生产能力、没现金流、没退路”的生活方式。

### 阶段 B：新东方名师与教材作者期（2000 年末—约 2007/08）

关键词：托福/GRE/GMAT、课堂表达、教材写作、教师人格。  
这一阶段奠定其表达风格：重定义、反废话、强调可执行。

### 阶段 C：方法论作者期（2007—2010s 前半）

关键词：《把时间当作朋友》、时间不可管理、心智成长、长期积累。  
这是其“母框架”形成期，几乎所有后续作品都能追溯到这里。

### 阶段 D：比特币学习者/持有者/布道者期（2011—2014）

关键词：Twitter 初见 Bitcoin、早期买入、暴涨暴跌、长期持有、世界银行叙事。  
这一阶段给了他超额收益经验，也塑造了他对市场、人性与认知错误的强烈判断口吻。

### 阶段 E：区块链投资人与争议高峰期（2015Q4—2018）

关键词：INBlockchain、MBA 项目、ICO 时代、录音风波、《韭菜的自我修养》。  
这是他最强势、也最危险的阶段：声望、影响力、争议、道德质疑同时上升。

### 阶段 F：边界收缩后的课程/社群主理人期（2019—至今）

关键词：讲课多于写作、富足人生践行群、思考/财富/学习等“真相”系列、微信视频号、定投与长期主义。  
这阶段的李笑来更像一个“认知生活方式设计者”，而不只是投资人。

## 仍需保留的不确定点 / 开放问题

1. **出生年份、出生地等基础传记项**  
   中文互联网二手资料常见统一说法，但我在本轮优先使用的第一手材料里还没看到足够硬的自证，因此本稿不写死。

2. **离开新东方的精确年份**  
   可确认 2000 年底前后加入，并“教了七年”；若精确到离职年月，需要补采访或简历类一手证据。

3. **《财富的真相》《思考的真相》《微信互联网平民创业》等近年书的精确出版时间**  
   官网能确认这些作品/配套课程存在，但没有在同一处给出统一、精确的出版年月。

4. **微信视频号与近年课程体系的启动时间**  
   官方页能证明“存在且仍在运营”，但起点年份、更新频率、影响规模尚未从第一手公开材料中完整重建。

5. **2018 录音争议的外部时间线**  
   当前能确定李笑来本人的回应文本与其自我理解；但若要把“泄露—传播—媒体放大—市场环境”的外部链条写扎实，仍应补权威媒体报道。

## 对最终 identity card / timeline section 的直接建议

- 会计专业出身，毕业后做过销售/批发；
- 2000 年底前后进入新东方，教书约 7 年；

- 2018 年因录音风波与“韭菜”标签成为高度争议人物，并以《韭菜的自我修养》作系统回应；
- 2019 年后更多讲课，2018/2019/2022 连续收缩投资边界；
- 后期主轴是课程/社群/视频号/定投与思考方法论产品。
