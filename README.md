# 李笑来.skill

> “认知升级不是选项，是你能不能活下来的底层游戏。”

**李笑来人物 Skill 的可运行认知操作系统版本。**

不是语录合集，不是观点拼贴。
是把李笑来“认知优先、价值逻辑、长期主义、负面清单决策”提炼成可执行模型。

</div>

---

## 目录结构

- `SKILL.md`：核心人物 Skill 文件。
- `scripts/`：辅助工具脚本，包括质量检查、字幕清洗、调研摘要。
- `references/`
  - `research/`：按著作、对话、表达、他者观点、决策、时间线拆分的调研文档。
  - `sources/`：已知书籍/文章/页面链接与素材记录。

---

## 说明

这个仓库是一个“可运行的李笑来认知框架”。

`SKILL.md` 的设计目标是：

- 把李笑来的核心心智模型抽象成可复用的决策逻辑；
- 用“回答工作流”避免凭感觉结论；
- 用“表达DNA”给输出固定的风格；
- 用“诚实边界”标注不可触碰的局限。

---

## 你能得到什么

- 4 个核心心智模型
- 6 条决策启发式
- 一套“李笑来式”问题分类与研究流程
- 可用于投资、创业、学习、风险判断的认知模板
- 参考来源与素材目录，便于后续迭代

## 使用示例

### 问：我该不该做一个区块链项目？

```
李笑来  ❯ 先别问“这个项目能不能火”，
          问“我懂不懂它的价值逻辑”。

          如果你连这个项目的底层需求、商业闭环、可复制的护城河都讲不清楚，
          你现在讨论的只是一个概念，一个故事。

          真实问题是：你能否承受它失败后的后果？
          你是否愿意把时间、声誉、资本和认知都押在这个假设上？

          结论：除非你能把它变成“我可以解释给一个完全不懂的人”，
          否则先不要去做。
```

> 完整的实战对话记录在 [`examples/`](examples/) 目录。

---

## 这个 Skill 是怎么造出来的

基于 [女娲.skill](https://github.com/alchaincyf/nuwa-skill) 的流程：

1. 明确人物与调研目标
2. 多维度并行采集素材（著作/对话/表达/他者/决策/时间线）
3. 提炼可执行心智模型与决策规则
4. 构建 Skill 并进行质量检查
